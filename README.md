# ISD Lab Service Desk

A premium, modern ticketing portal for the **Intelligent Systems Design Lab (ISD Lab)**. This website serves as a streamlined interface for submitting technical requests, reporting issues, and tracking support tickets via ClickUp integration.

## 🚀 Features

- **Premium Aesthetics**: High-tech laboratory-themed design with a custom-built dark mode.
- **Glassmorphism UI**: Modern translucent interfaces with smooth backdrop blurs and glows.
- **ClickUp Integration**: Seamlessly embedded ClickUp forms with dynamic height adjustment for a native-like experience.
- **Full-Page Scrolling**: Optimized to prevent internal iframe scrollbars, allowing for a single, unified scrolling experience.
- **Brand Cohesion**: Fully aligned with the official ISD Lab logo and color palette.
- **Responsive Design**: Mobile-first approach ensuring portability across all devices.

## 🛠 Technology Stack

- **Core**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Design System)
- **Integration**: ClickUp Embed API
- **Fonts**: Google Fonts (Outfit)

## 📂 Project Structure

- `index.html`: The main entry point and portal structure.
- `style.css`: Custom styling, animations, and responsiveness.
- `logo.png`: Official ISD Lab branding.
- `bg.png`: Technical laboratory background asset.

## 🌐 Getting Started

To run the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Intelligent-Systems-Design-Lab-SRM/service-desk.git
   ```
2. Open `index.html` in any modern web browser.

Alternatively, serve it using a simple HTTP server:
```bash
python3 -m http.server 8000
```
Then visit `http://localhost:8000`.
